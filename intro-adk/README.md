# ADK

Investigation of the [Agent Development Kit Python Quickstart](https://adk.dev/get-started/python/). Runs a local agent without Agent Engine — no GCP deployment required.

## Setup

```bash
pyenv virtualenv 3.14 adk
pyenv activate adk
pip install -r requirements.txt
adk create my_agent
```

The `my_agent` directory was generated by `adk create` and is already present.

## Usage

Export required environment variables:

```bash
export GOOGLE_GENAI_USE_ENTERPRISE=1
export GOOGLE_CLOUD_PROJECT=system-alexb-art-ed9d
export GOOGLE_CLOUD_LOCATION=europe-west2
```

### CLI

```bash
adk run my_agent
```

### Web UI

```bash
adk web --port 8080
```

Then visit http://127.0.0.1:8080
